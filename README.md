# MDFNet
The code with paper "MDFNet: A Single-Stage Detection Network for Ship Instance Segmentation".
Instance segmentation technology plays a crucial role in visual perception in maritime scenarios, particularly in fields like unmanned vessel development where efficient and accurate segmentation is highly demanded. This paper proposes a new single-stage instance segmentation network, MDFNet, based on the YOLOv8 instance segmentation model. In the backbone network, we introduce the CpnMvitbv3 module, and in the neck network, we adopt the DODConv module. Additionally, a novel loss function—Focal-SIoU—is designed, significantly improving the accuracy of ship instance segmentation tasks. MDFNet also demonstrates outstanding performance in balancing precision and recall, effectively reducing false positives and false negatives. It achieves a high detection accuracy rate, a low missed detection rate, and enhanced recognition capabilities for small ship targets.

Through extensive experiments, MDFNet was compared with mainstream instance segmentation models on the SSDD and MarShipInsSeg datasets. Results show that MDFNet achieves AP scores of 63.8 and 49.4 on these datasets, respectively, outperforming the second-best models, Hybrid Task Cascade and Yolact, by 2.7 and 5.8 percentage points. Experimental results indicate that MDFNet not only achieves significant improvements in ship segmentation accuracy but also exhibits strong generalization capabilities, making it adaptable to different datasets and environmental conditions. This provides a new and effective solution for ship monitoring and identification tasks. The code is available at: https://github.com/RainbowSugar1/MDFNet.
