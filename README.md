# MDFNet
The code with paper "MDFNet: A Single-Stage Ship Instance Segmentation Network for SAR and Visible Light Images".

Instance segmentation technology plays a vital role in visual perception in maritime environments, particularly in fields like unmanned ship development, which demand highly efficient segmentation. This paper proposes a novel single-stage instance segmentation network, MDFNet, based on the YOLOv8 instance segmentation model.In the backbone network, the CpnMvitbv3 module incorporating transformer mechanisms is introduced, while the neck network employs a dynamic convolution module (DODConv). Additionally, a new loss function, Focal-SIoU, is designed to significantly enhance segmentation accuracy for ship instance detection tasks. MDFNet achieves an exceptional balance between precision and recall, effectively reducing false positives and false negatives, thereby improving detection accuracy and its capability to identify small ship targets.Through extensive experiments, MDFNet is compared with mainstream instance segmentation models on the SSDD (SAR image dataset) and MarShipInsSeg (visible-light image dataset). Results demonstrate that MDFNet achieves an AP accuracy of 63.8 on the SSDD dataset, surpassing the Hybrid Task Cascade and Yolact models by 2.7 and 5.8 percentage points, respectively. MDFNet particularly excels in SAR image performance, providing a robust solution for ship monitoring and recognition.Experimental findings reveal that MDFNet not only delivers significant improvements in ship segmentation accuracy but also exhibits strong generalization capabilities, adapting effectively to different datasets and environmental conditions. Its outstanding performance in SAR images makes it a promising tool for maritime surveillance and identification tasks.The code is available at ：https://github.com/RainbowSugar1/MDFNet.
